Guide d'Installation

Suivez ces étapes pour installer votre blog Laravel

1

Activer l'extension ZIP

Pourquoi ? Pour accélérer l'installation de 10x !

Important : Sans ZIP, l'installation prend 15-20 minutes. Avec ZIP : 2-3 minutes !

Étapes :

  • Ouvrir XAMPP Control Panel
  • Arrêter Apache (bouton "Stop")
  • Cliquer sur "Config" → "PHP (php.ini)"
  • Chercher ";extension=zip" (Ctrl+F)
  • Enlever le ";" pour avoir "extension=zip"
  • Sauvegarder (Ctrl+S)
  • Redémarrer Apache (bouton "Start")
Astuce : Consultez GUIDE_ACTIVATION_ZIP.md pour un guide détaillé avec captures d'écran
2

Vérifier l'activation de ZIP

Ouvrez un nouveau terminal et tapez :

php -m | findstr zip
Si vous voyez "zip" dans la liste, c'est bon ! Passez à l'étape suivante.
Si "zip" n'apparaît pas, recommencez l'étape 1 et assurez-vous d'avoir redémarré Apache.
3

Réinstaller Composer

Méthode Simple :

Double-cliquez sur le fichier :

reinstaller-composer.bat

OU en ligne de commande :

cd blog-laravel reinstaller-composer.bat
L'installation prendra 2-3 minutes avec ZIP activé.
4

Créer la base de données

Ouvrez phpMyAdmin ou votre client MySQL et exécutez :

CREATE DATABASE blog_laravel CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
5

Configurer le projet

Dans le dossier blog-laravel, exécutez :

setup.bat

Ce script va :

  • Générer la clé d'application
  • Exécuter les migrations
  • Créer les utilisateurs de test
  • Créer le lien pour les images
6

Lancer le serveur

Exécutez :

php artisan serve
Le serveur démarre sur http://localhost:8000
7

Ouvrir dans le navigateur

Ouvrez votre navigateur et allez sur :

http://localhost:8000
