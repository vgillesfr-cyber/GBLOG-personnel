Étapes d'Installation

1. Activer l'extension ZIP (Recommandé)

Pour accélérer l'installation :

1. Ouvrir C:\xampp\php\php.ini 2. Chercher ;extension=zip 3. Enlever le ; pour avoir : extension=zip 4. Sauvegarder et redémarrer Apache

2. Créer la Base de Données

CREATE DATABASE blog_laravel;

3. Configurer le Projet

Une fois l'installation de Composer terminée :

setup.bat

4. Lancer le Serveur

php artisan serve

Puis ouvrir : http://localhost:8000
