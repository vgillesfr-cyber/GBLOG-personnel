Votre blog Laravel a été transformé en blog personnel avec :
  ✅ Système de rôles (Propriétaire/Visiteur)
  ✅ Support multi-média (Images, Vidéos, Documents)
  ✅ Système de likes ❤️
  ✅ Profil personnalisé
  ✅ Page "À propos" dynamique


═══════════════════════════════════════════════════════════════
📋 ÉTAPES POUR DÉMARRER
═══════════════════════════════════════════════════════════════

1️⃣  APPLIQUER LA MISE À JOUR
   ────────────────────────────────────────────────────────────
   Double-cliquez sur : APPLIQUER_MISE_A_JOUR.bat
   
   Ce script va :
   • Appliquer les nouvelles migrations
   • Créer le lien de stockage pour les médias
   • Nettoyer le cache


2️⃣  CRÉER LES UTILISATEURS DE TEST
   ────────────────────────────────────────────────────────────
   Ouvrez un terminal et exécutez :
   > php artisan db:seed --class=AdminSeeder


3️⃣  DÉMARRER LE SERVEUR
   ────────────────────────────────────────────────────────────
   > php artisan serve
   
   Puis ouvrez : http://127.0.0.1:8000
